# Project Brand Guardrails

## Principios

- La marca del proyecto manda sobre paletas genericas.
- El adapter no inventa tokens si existe fuente operativa.
- Si faltan tokens, devuelve BLOCKED_BRAND_SOURCE_MISSING.
- Si el flujo esta EN_REVISION, el mockup debe mostrar estado revisable.

## Regla Overall

Para Overall / OverCash:

- usar OverCash visible;
- usar blue/indigo/violet como primarios;
- usar success/warning/error solo semanticamente;
- no usar paleta Marketplace LF;
- no usar dark mode salvo fuente autorizada;
- no usar gamificacion persuasiva si screen_visual_specs la bloquea.

## QA

Readback debe listar:

- design_system_code;
- tokens usados;
- pantallas mockupeadas;
- specs visuales aplicadas;
- bloqueos o desviaciones.
