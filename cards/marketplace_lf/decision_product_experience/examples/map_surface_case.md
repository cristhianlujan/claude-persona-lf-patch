# Example — Ruta de Claridad Map Surface Decision Case

Status: SANDBOX_EXAMPLE / READ_ONLY

## Decision question

Evaluate the recommended surface style for the Ruta de Claridad map/path in MarketPlace Libertad Financiera.

## Options

1. Minimal organic warm surface.
2. Light isometric surface.
3. Highly visual 3D game-like map.
4. Flat corporate process diagram.

## Evaluation focus

- Clarity of path.
- Reduced cognitive load.
- Fintech credibility.
- Calm progress perception.
- Avoiding fantasy, game excess and infantilization.
- Avoiding cold corporate diagram.
- Supporting Juan Digital and mission active state without visual competition.

## Expected direction

Prefer a minimal organic warm surface with very light depth only if needed. Use isometric lightly only if it improves affordance without becoming game-like. Block high 3D fantasy/game maps. Block cold corporate diagrams if they reduce emotional guidance.

`MARKETPLACE_DECISION_RECOMMENDATION` if source context and target visual constraints are sufficient.
