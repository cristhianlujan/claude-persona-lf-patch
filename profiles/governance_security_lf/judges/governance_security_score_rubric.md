# Governance Security Score Rubric

Total: 25 points. Minimum PASS: 22/25 and no hard fail.

| Criterion | Points |
|---|---:|
| Router/source-of-truth enforcement | 5 |
| Scope and actor boundary control | 5 |
| Unauthorized write/state promotion blocking | 5 |
| Evidence/readback quality | 5 |
| Closed output modes and handoff clarity | 5 |

## Automatic Fail
Any runtime enablement, automatic impact enablement, VALIDATED/VIGENTE/APPROVED declaration, manual protocol execution, or incomplete pack closure without evidence is FAIL regardless of score.
