# Page Contracts

L1 Cover: title, subtitle, state, source, date, version.
L2 Summary: four cards, verdict, next action.
L3 Flow: max seven nodes, phase labels, status chips.
L4 Screens: four to six cards with action, risk and status.
L5 Risks: short table with severity and control.
L6 Dependencies: documents and systems separated.
L7 QA: rendered pages, reviewed pages, issues, fixes, pending.

Fail if page is only text, too dense, unreadable, or without visual evidence.
