# Example input brief

## Solicitud
Crear PDF ejecutivo para revisar un design system LF.

## Audiencia
Owner / direccion / equipo operativo.

## Tipo de PDF
Brandbook + sistema de componentes.

## Estilo
Fintech corporativo, premium, sobrio, visual, no basico.

## Fuentes
- Imagenes de referencia provistas por el usuario.
- Paleta derivada visualmente.
- Decisiones simples tomadas por el asistente.

## Restricciones
- No inventar datos no visibles.
- Registrar supuestos.
- Entregar PDF revisable.
- Incluir paleta, tipografia, componentes, usos y no-usos.

## QA esperado
- Portada profesional.
- Indice.
- Componentes visuales.
- Tablas legibles.
- Semaforos y flujos cuando aplique.
- Readback de paginas y pendientes.
