# Report Exec 16x9 Template

Use for executive PDF, board review, proposal or flow report.

Page map:
1. Cover with status chips.
2. Executive summary with four cards.
3. Flow map with max seven nodes.
4. Key findings as cards.
5. Risks and controls as short table.
6. Decision board.
7. Next actions and evidence.

Rules:
- Landscape 16:9.
- Safe margin minimum 0.5 inch.
- One main message per page.
- Max six cards per page.
- Max eight table rows per page.
- Body text minimum 10 pt.
- Footer: version, state and page.

Fail if the page looks like plain memo, dense table dump, or has unreadable flow.
