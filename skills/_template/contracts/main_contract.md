# Main Contract

## Contract

The pack must produce structured, schema-valid, reusable outputs with evidence and handoff readiness.

## Required output fields

- `status`
- `pack_type`
- `deliverable_created`
- `evidence_map`
- `blocking_codes`
- `next_gate`

## Invalid output

A prose-only prompt, generic checklist or undocumented recommendation is not acceptable.
